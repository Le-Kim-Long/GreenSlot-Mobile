# GreenSlot Mobile

Ứng dụng React Native cho nền tảng thuê vườn canh tác thẳng đứng **GreenSlot**, đồng bộ UI/workflow với `GreenSlot-FrontEnd`.

## Công nghệ

- React Native (Expo SDK 52)
- TypeScript
- React Navigation
- Axios + JWT
- Font Inter (giống web)
- Design system màu xanh `#16a34a` / nền `#f0faf4`

## Chạy ứng dụng

```bash
npm install
npx expo start
```

## Tính năng

- Đăng nhập / Đăng ký / Quên mật khẩu
- Khám phá & thuê ô vườn (VNPay)
- Dashboard khách hàng
- Giám sát IoT
- Dịch vụ chăm sóc
- Lịch sử thanh toán
- Nhiệm vụ nhân viên vườn
